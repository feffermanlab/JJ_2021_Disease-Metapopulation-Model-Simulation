Note: you need to download and install Matlab with statistical toolbox to be able to perform the analyses.

"base.m": provides the backbone of Matlab code, indicating how disease can spread with host migration (random migration) in a well-connected network. 

The detailed code to generate each figure at this publication were provided as the code name.

For example, "Figure_2.m" can generate the initial excel csv data of Figure 2 from matlab, and "Figure_2.r" uses the initial data to plot the corresponding figure.   

"Figure_5 with S4.m" can generate the initial excel csv data of Figure 5, while "Figure_5 with S4.r" uses the csv data to plot Figure. 5.
